### 本地技术架构
| 层级 | 技术选型 | 核心职责 | 代码量占比 |
|------|------|------|------|
| 桌面外壳层 | C++ + CEF（Chromium 嵌入式框架） | 原生窗口渲染、进程守护、系统权限桥接（托盘 / 快捷键 / 文件拖拽）、前端静态资源加载 | 8%~10% |
| 前端界面层 | 原生 HTML/CSS/JS（规划迁移至 Vue） | 聊天界面、功能面板、交互渲染、流式打字机效果 | 10%~15% |
| 业务核心层 | Python 3.11 + FastAPI | Agent 调度、API 调用封装、本地工具集执行、数据读写、模式切换逻辑 | 60%~65% |
| 本地存储层 | SQLite + FAISS 向量库 | 聊天历史、用户配置、API 密钥、本地知识库向量索引 | 包含在 Python 层 |

### 云端技术架构
| 层级 | 技术选型 | 核心职责 | 代码量占比 |
|------|------|------|------|
| 接口层 | Python FastAPI | 账号鉴权、API 代理转发、额度扣减、数据同步 | 10%~12% |
| 存储层 | MySQL + Redis | 用户信息、余额与订单、调用日志、限流缓存 | 包含在云端层 |
